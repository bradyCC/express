## 代码结构
```
|---bin                        相关运行脚本
    |---www                    项目启动脚本
|---libs                       公共函数                  
|---public                     静态资源文件

    |---javascripts            javascript 
    |---stylesheets            css     
    |---upload                 图片上传                  
|---routes                     路由
|---views                      视图模版                     
app.js                         应用核心配置文件(入口文件)
README.md                      介绍
```
